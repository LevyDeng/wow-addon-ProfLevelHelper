# ProfLevelHelper

时光服冲专业助手插件。

---

## 原理

获取拍卖行所有价格数据，根据价格切片算出该价格下最省钱的冲专业路线。

## 使用方法

1. 插件全关，只留 **ala 商业技能助手** 和本插件；若需使用附魔回血功能，还需开启 **Auctionator** 插件。
2. 打开拍卖行页面，点击上方的 **「ProfLevelHelper 扫描」** 按钮。
3. 等待扫描完成（会有点卡，属正常现象）。
4. 扫描完成后，打开专业技能页面，点击 **「冲点助手」** 按钮即可看到冲点规划。
5. 可在选项中修改配置，以适合自己的计算方式。
